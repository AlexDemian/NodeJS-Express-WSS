# NodeJS-Express-WSS

### Description

WSS-on-NodeJS playground. <br>
Simple one-channel chat app based on websockets server running on NodeJS. <br>
Allows to publish text and images to the channel. <br>

### Infrastructure

NodeJS/Express app <br>
Websocket server <br>

### Installation & Startup

```
npm i
npm run dev 
# or
npm run watch
```
